## phpstorm 使用总结

### 快捷键
command + e 列出最近打开的文件

ctrl + / 行注释
ctrl + shift + / 块注释和取消注释

shift + shift 全局搜索
command + n 根据类名查找
command + shift + n 根据文件查找
command + shift + alt + n 根据符号查找

command + f 文件内查找
command + r 文件内替换
command + g 跳转到指定行
command + b 跳转到变量声明处

command + alt + l 格式化代码


shift + F6 重命名方法

command + d 复制当前行
command + y 删除当前
command + j 调用输出模板

command + F12 显示当前文件中的方法列表
